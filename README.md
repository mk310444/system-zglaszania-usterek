# System zgłaszania usterek

Prosta aplikacja konsolowa napisana w języku C++.

## Opis

Program służy do zgłaszania i obsługi usterek w budynku, np. w szkole, firmie lub akademiku.

## Funkcje

- dodawanie zgłoszeń,
- wyświetlanie listy zgłoszeń,
- wyszukiwanie zgłoszenia po ID,
- zmiana statusu zgłoszenia,
- usuwanie zgłoszeń,
- wyświetlanie statystyk.

## Technologie

- C++
- aplikacja konsolowa
